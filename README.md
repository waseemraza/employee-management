# README #


This repository is to showcase different technical concepts:

* Basic spring boot project setup.
* build.gradle configuration.
* configure liquibase.
* configure integration test using com.softeq.gradle.itest gradle plugin.
* gradle plugin java-test-fixtures to create centralized test data to be used by unit test and integration test.
* configure docker compose supported by spring boot internally through spring-boot-docker-compose dependency.
